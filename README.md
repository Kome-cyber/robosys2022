# plusコマンド
[![test](https://github.com/Kome-cyber/robosys2022/actions/workflows/test.yml/badge.svg)](https://github.com/Kome-cyber/robosys2022/actions/workflows/test.yml)

* 標準入力から読み込んだ数字を足す。
```
seq 10 | ./plus
55
```
## 必要なソフトウェア
* Python
 * テスト済みバージョン: 3.7～3.10
## テスト環境
* Ubuntu 22.04.1 LTS
